[TO RUN sudoku.hs]
Unix:
  1. stack ghci sudoku.hs
  2. main

Windows 10:
  1. ghci
  2. :l sudoku.hs
  3. main

==> TO LOAD TYPE:
1. :l

==> TO RELOAD TYPE:
1. :r

==> TO QUIT TYPE:
1. :q

*** NOTE ***
- To edit the input board, change line 98 to be set to the desired input board

[TO RUN sudoku.js]
1. open index.html

==> TO SEE BACKTRACKING IN ACTION:
1. Open the Console (fn + F12 on my MacBook Pro)

*** NOTE ***
- To edit the input board, comment out the current input board and uncomment the
  desired input board
